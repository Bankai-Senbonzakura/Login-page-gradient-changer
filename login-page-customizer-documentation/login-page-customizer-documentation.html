Login Page Customizer Plugin Documentation

1. Introduction

Overview: The Login Page Customizer plugin allows you to easily customize the background color of your WordPress login page. Whether you prefer a simple solid color or a stylish gradient, this plugin provides intuitive options to personalize your login experience.

Key Features:

  • Customize the login page background with gradient backgrounds.
  • Choose from two types of gradients: linear and radial.
  • Simple and straightforward interface for quick customization.

2. Installation

Requirements:

  • WordPress Version Required: 4.0 or higher
  • Compatible up to: 4.8.25

Installation Steps:

  1. Manual Installation:
    1. Download the plugin folder from GitHub.
    2. Upload the plugin folder to the wp-content/plugins/ directory of your WordPress installation.
    3. Activate the plugin through the 'Plugins' menu in WordPress.
  2. WordPress Admin Panel:
    1. Download the plugin as a zip file.
    2. Go to your WordPress admin panel.
    3. Navigate to Plugins > Add New.
    4. Click on the 'Upload Plugin' button and choose the zip file.
    5. Activate the plugin.

3. Getting Started

Accessing the Plugin:

  • Once activated, go to the WordPress admin dashboard.
  • Navigate to the 'Login Customizer' menu.

Initial Setup:

  • You will find options to customize the login page background color.
  • Adjust the colors and gradient type as desired.

4. Customization Options

Gradient Backgrounds:

  • Customize the login page background using gradient colors.
  • Choose from linear or radial gradients.

Color Pickers:

  • Use integrated color pickers to select custom colors for gradient backgrounds.

Gradient Types:

  • Linear Gradient: Creates a gradient that transitions in a straight line.
  • Radial Gradient: Creates a circular gradient emanating from a center point.

Gradient Directions:

  • Select the direction of the gradient:
  • To Right, To Left
  • To Bottom, To Top
  • To Bottom Right, To Bottom Left
  • To Top Right, To Top Left

5. Troubleshooting and FAQ

Common Issues:

  • Issue: Background does not update after applying changes.
  • Solution: Ensure you have internet access for the plugin to function properly.

Frequently Asked Questions:

  • Q: How do I know if my changes have applied?
  • A: Visit your login page to see the updated background.
